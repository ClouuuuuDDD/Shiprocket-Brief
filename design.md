# 🔧 **Visual System Enhancements (ADDED — IMPORTANT)**

### **1. Charts & Data Visualization Style**

Your earlier system was missing depth here — this fixes it.

* Charts should feel like:

  > “Printed financial newspaper meets dashboard terminal”

#### Rules:

* No rounded bars, ever
* Use:

  * **Hard rectangles**
  * **Outlined bars (2–4px black stroke)**
* Labels directly on elements (no legends)
* Use mono font for numbers

#### Chart Types:

* **Bar Charts:**

  * Thick, blocky
  * Alternating fill (solid vs outlined)

* **Pie Charts:**

  * Avoid if possible
  * If used → sharp segmented wedges, no gradients

* **Framework Diagrams (preferred):**

  * Boxes + arrows
  * Strong borders
  * Color-coded meaning

---

### **2. Card System (Replaces “UI blocks”)**

* No soft cards

* Use:

  * Hard containers
  * 4px black borders
  * Hard shadows (`6px 6px 0px #000`)

* Cards should feel:

  > Like printed cutouts pasted onto a board
